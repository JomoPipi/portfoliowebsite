Coding Projects

  • NUNISYNTH

    NUNISYNTH is a graphical modular audio synthesizer based on the Web Audio API. Its original purpose was to facilitate exploring and experimenting with the API, but it evolved into something more.

    • It's been through several iterations of UI refinements. Even though it gives you fine-grained audio control, it lets users create intricate things with relatively few steps.
    • Auxiliary utilities (for example: scale builders, sample-generating formulas, visualizers, keyboarding, recording, importing, exporting, keyboard shortcuts, saving, loading, and more) were added.
    • A number of custom Web Audio nodes were added, including Automation, Sample Sequencer, Gate Sequencer, Sample Piano Roll, Mono Piano Roll, Recorder, Module.
    • Modules play a big role. They are basically nested graphs. Users can save their own custom modules and reuse them in any project. A handful of helpful modules are provided by default.
